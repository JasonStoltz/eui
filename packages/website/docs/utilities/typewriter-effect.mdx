## Basic usage

The `useTypewriterEffect` hook creates a typewriter animation that types out text character by character. It's perfect for creating engaging user interfaces, loading states, or conversational interfaces.

<Playground
  config={{
    imports: {
      '@elastic/eui': {
        named: ['useTypewriterEffect', 'EuiPanel', 'EuiText', 'EuiButton', 'EuiFlexGroup', 'EuiFlexItem']
      }
    },
    scope: {
      React: React,
    }
  }}
  code={`const TypewriterDemo = () => {
  const { displayText, isTyping } = useTypewriterEffect({
    text: "Welcome to EUI!",
    typingSpeed: 50,
  });

  return (
    <EuiPanel paddingSize="l">
      <EuiText>
        <div style={{ 
          fontSize: '18px', 
          fontFamily: 'monospace',
          minHeight: '24px',
          border: '1px solid #d3dae6',
          padding: '8px 12px',
          borderRadius: '4px',
          backgroundColor: '#f8f9fa'
        }}>
          {displayText}
          {isTyping && <span style={{ animation: 'blink 1s infinite' }}>|</span>}
        </div>
      </EuiText>
    </EuiPanel>
  );
};

render(<TypewriterDemo />);`}
/>

## API

### Parameters

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | - | The text to type out |
| `typingSpeed` | `number` | `50` | Speed of typing in milliseconds between each character |
| `enabled` | `boolean` | `true` | Whether the typewriter effect is enabled |
| `respectReducedMotion` | `boolean` | `true` | Whether to respect user's reduced motion preferences |
| `announceChanges` | `boolean` | `false` | Whether to announce text changes to screen readers |
| `onStart` | `() => void` | - | Callback fired when typing starts |
| `onComplete` | `() => void` | - | Callback fired when typing is complete |

### Returns

| Name | Type | Description |
|------|------|-------------|
| `displayText` | `string` | The current displayed text |
| `isTyping` | `boolean` | Whether the typewriter effect is currently typing |
| `isComplete` | `boolean` | Whether the typing is complete |
| `respectsReducedMotion` | `boolean` | Whether the effect respects reduced motion preferences |
| `start` | `() => void` | Manually start the typewriter effect |
| `stop` | `() => void` | Manually stop the typewriter effect |
| `reset` | `() => void` | Reset the typewriter effect |

## Accessibility

The `useTypewriterEffect` hook includes several accessibility features to ensure a good experience for all users:

### Reduced Motion Support

The hook automatically respects the user's `prefers-reduced-motion` setting by default. When this preference is detected, the typewriter effect is disabled and text appears immediately.

```typescript
const { displayText, respectsReducedMotion } = useTypewriterEffect({
  text: "Hello, world!",
  respectReducedMotion: true, // Default: true
});

// Check if reduced motion is being respected
if (respectsReducedMotion) {
  console.log("User prefers reduced motion, effect disabled");
}
```

### Screen Reader Announcements

You can optionally enable screen reader announcements for typing events:

```typescript
const { displayText } = useTypewriterEffect({
  text: "Important message",
  announceChanges: true, // Default: false
});
```

**Note**: Screen reader announcements are disabled by default to avoid overwhelming users with repeated announcements as each character appears. Only enable this for important status changes.

### Best Practices

- **Respect user preferences**: Always keep `respectReducedMotion: true` (default)
- **Minimize announcements**: Only use `announceChanges: true` for critical status updates
- **Provide alternatives**: Consider providing a way for users to skip or disable animations
- **Test with assistive technology**: Verify the experience works well with screen readers and other tools

## Implementation details

The hook uses `setTimeout` to create the typing effect and automatically cleans up timers when the component unmounts or when the text changes. It tracks the current typing state and provides manual controls for advanced use cases.
